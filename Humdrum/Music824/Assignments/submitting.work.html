How to Submit an Assignment for Grading

Assignments for Music 824 should normally be submitted electronically. This document describes how to submit assignments. Please follow these instructions carefully.

If you encounter any difficulty, you are welcome to submit a printed or handwritten assignment by placing it in the envelop outside Dr. Huron's office (Mershon Auditorium, Room 502).

  • From the webpage for the assignment, cut and paste the text into a plain-text file.

  • Edit your copy of the file using a text editor such as notepad, pico, vi or emacs. To the right of the "NAME:" prompt, type your name.

  • As you work through the assignment, add the appropriate responses to the text file. For example, an assignment might contain the following question:

  1. Is the file "syrinx" a Humdrum file?

    COMMAND:
    ANSWER:
You might answer the question by adding suitable text such as the following (shown in bold):
  1. Is the file "syrinx" a Humdrum file?

    COMMAND:   humdrum syrinx
    ANSWER:       yes

  • When you have answered all of the questions in the assignment you can submit your work via e-mail. It is preferable to e-mail your work from your own userid in order to avoid possible confusion about your identity.
