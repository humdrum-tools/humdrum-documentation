Humdrum Exercise #6 - Answers

All of the following commands assume that you are located in the appropriate score directory.

    2.
    How many measures are there in Corelli's Opus 1, No. 11, first movement?
    COMMAND:     census -k op01n11a.krn
    COMMAND:     grep -c ^=[^=]' op01n11a.krn
    ANSWER:         18
    Notice that the grep command would fail to identify barlines that consist of a single equals sign on a line by itself. As a result, using census is better than using grep.
